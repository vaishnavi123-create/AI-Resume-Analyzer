# AI Resume Analyzer

An AI-powered Resume Analyzer built with Python and Flask that helps users upload resumes, extract key information, analyze skills, and gain insights to improve resume quality.

## Features

- 📄 Upload resumes (PDF/DOCX)
- 🔍 Extract resume text
- 🧠 AI-powered resume analysis
- 💼 Skill extraction
- 📊 Resume insights and recommendations
- 🌐 Simple and user-friendly web interface

##  Technologies Used

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- NLP (Planned)
